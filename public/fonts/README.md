# Giftologi Custom Fonts

Place the following font files in this directory:

- `DidotBold.woff2` (or `.woff` / `.otf` / `.ttf`) — used for **headings**
- `BrasleySemibold.woff2` (or `.woff` / `.otf` / `.ttf`) — used for **subheadings**
- `BrasleyMedium.woff2` (or `.woff` / `.otf` / `.ttf`) — used for **paragraphs/body text**

After adding the files, update the `src` paths in `app/layout.js` if the file extensions differ from `.woff2`.
